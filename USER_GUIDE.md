# Guia de Uso

O ArqFlow ainda esta em fundacao tecnica. O uso final sera documentado a cada modulo entregue.

## Fluxo previsto

1. Registrar cliente.
2. Criar briefing.
3. Criar orcamento.
4. Aprovar ou recusar orcamento.
5. Converter orcamento aprovado em projeto.
6. Acompanhar etapas.
7. Registrar financeiro, tarefas, visitas e documentos.
8. Finalizar projeto.
