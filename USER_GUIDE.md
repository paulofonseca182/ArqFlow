# Guia de Uso

O ArqFlow ainda esta em fundacao tecnica. O uso final sera documentado a cada modulo entregue.

## Fluxo previsto

1. Registrar cliente.
2. Criar orcamento.
3. Aprovar ou recusar orcamento.
4. Converter orcamento aprovado em projeto.
5. Acompanhar etapas.
6. Registrar financeiro, tarefas e visitas.
7. Finalizar projeto.
