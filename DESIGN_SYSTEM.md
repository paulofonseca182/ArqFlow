# Design System

## Direcao visual

O ArqFlow segue uma interface dark premium:

- Minimalista
- Sofisticada
- Arquitetonica
- Urbana
- Profissional
- Linhas finas
- Cards discretos
- Bastante respiro visual

## Paleta

- Preto absoluto: `#000000`
- Grafite profundo: `#0B0B0B`
- Card escuro: `#141414`
- Card secundario: `#1A1A1A`
- Borda sutil: `#2A2A2A`
- Texto principal: `#F5F5F5`
- Texto secundario: `#A8A8A8`
- Texto suave: `#6F6F6F`
- Azul institucional: `#103253`
- Bronze destaque: `#B87333`
- Verde sucesso: `#3BA66B`
- Amarelo alerta: `#D6A84F`
- Vermelho atraso: `#D95C5C`

## Componentes base

- `Sidebar`
- `Header`
- `PageWrapper`
- `Button`
- `Input`
- `Select`
- `Textarea`
- `Card`
- `Table`
- `Badge`
- `Modal`
- `DeleteModal`
- `ProgressBar`
- `StatCard`
- `EmptyState`
- `LoadingState`

## Regras

- Status aparecem como badges.
- Exclusoes usam modal.
- Formularios usam componentes reutilizaveis.
- Tabelas devem aceitar filtros simples no topo da tela.
- A sidebar mantem a marca ArqFlow no topo.
